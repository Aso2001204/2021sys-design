```uml
@startuml
[*] -> トップページ
トップページ : do/おすすめ商品を表示
会員情報 --> トップページ : トップページをクリック
ログイン --> トップページ : トップページをクリック
トップページ --> トップページ : ログアウトをクリック
トップページ -left-> ログイン : ログインボタンをクリック
ログイン -left-> 会員情報 : ログイン成功時に遷移
ログイン : entry/ID-Passwordを入力
ログイン : do/ログイン認証
トップページ --> カート : カートをクリック
商品詳細 --> カート : カートに入れるをクリック
商品詳細 --> トップページ : 戻るをクリック
商品詳細 : do/商品説明などを表示
カート -up-> トップページ : トップページをクリック
カート -> 商品詳細 : 商品をクリック

state カート{
 [*] --> カート
 カート -> 注文 : 注文をクリック
 カート -> カート : 数量を変更/削除
 注文 --> カート : カートをクリック
 state 注文{
  [*] -> 商品詳細
  商品詳細 -> 住所等入力画面 : 次へをクリック
  住所等入力画面 -> 支払情報入力画面 : 次へをクリック
  支払情報入力画面 -> 完了画面 : 完了ボタンをクリック
  完了ボタン --> トップページ : トップページをクリック
 }
}
@enduml
```
