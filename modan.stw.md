```uml
@startuml
[*] -> トップページ
トップページ -left-> ログイン : ログインボタンをクリック
ログイン : entry/ID,PASSWORDを入力
ログイン : do/ログイン認証
ログイン --> トップページ : 戻るをクリック
トップページ -left-> about : aboutボタンをクリック
about : サイトの詳細を表示
about --> トップページ : 戻るをクリック
トップページ -> checkout : checkoutをクリック
checkout : カートの中を表示
トップページ -> product : productをクリック
product : おすすめを表示
product --> トップページ : 戻るをクリック
product -> 商品詳細 : 商品をクリック
商品詳細 : do/商品詳細を表示
商品詳細 -> checkout : カートに商品を追加
商品詳細 --> トップページ : 戻るをクリック


state checkout内 {
 [*] --> カート
 カート -> 注文 : 注文をクリック
 カート -> カート : 数量を変更/商品を削除
 注文 --> カート : カートをクリック
 state 注文 {
  [*] --> 送付先入力
  送付先入力 -> 送付先入力内容確認 : 入力内容確認をクリック
  送付先入力内容確認 -> 送付先入力 : 修正をクリック
  送付先入力内容確認 -> 購入完了 : 注文確定をクリック 
  購入完了 : 注文内容詳細を表示する
 }
}
@enduml
```
