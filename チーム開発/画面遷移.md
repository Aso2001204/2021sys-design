```uml
@startuml
[*]      -down->   会員認証:会員印象をする
会員認証  -down->   会員認証完了: ログイン成功時
会員認証完了 -down->トップページ
会員認証: entory/ID・Passwordを入力
会員認証: do/ログイン認証
会員認証  -right->   会員登録: 会員登録ボタンをクリック
会員認証      ->     会員メニュー:「パスワードを忘れ方はこちらへ」をクリック
会員登録: 新規会員登録用
トップページ      -right->   会員情報変更: 会員情報変更をクリック
会員情報変更  ->    トップページ: 戻るをクリック
トップページ       -down->  カート内: カートをクリック
トップページ       -up->    ログアウト: ログアウトをクリック
トップページ       -left->  お気に入り: お気に入りをクリック
トップページ     -down->   検索画面: 検索ボタンをクリック
検索画面   ->      会員メニュー
検索画面: 結果一覧を表示


state 会員登録{
 [*]     ->  入力画面
 会員登録画面     ->  確認画面:「確認画面」へをクリック
 確認画面    ->   登録完了画面:「送信」をクリック
 確認画面    ->       会員登録画面:「戻る」をクリック
 登録完了画面 ->     トップページ:「トップページ」に戻るをクリック
}

state 会員メニュー{
 [*]     ->  入力
 入力     ->  確認
 確認     ->  メール確認
 メール確認 -> 登録完了
}

state カート内{
 [*]    -->    カート
 カート  ->    レジに進む
 レジに進む  ->  配送先確認
 配送先確認 ->   支払方法確認
 支払方法確認 -> 注文確認
 注文確認 ->  注文完了
}

@enduml
```
