```uml
@startuml
[*]      -down->   ログイン 
ログイン  -right->   会員登録: 会員登録ボタンをクリック
会員登録: 新規会員登録用
ログイン  -down->   TOP: ログイン成功時
ログイン: entory/ID・Passwordを入力
ログイン: do/ログイン認証
TOP      -right->   会員情報変更: 会員情報変更をクリック
会員登録  ->       TOP: 戻るをクリック
TOP       -down->  カート内: カートをクリック
TOP       -up->    ログアウト: ログアウトをクリック
TOP       -left->  お気に入り: お気に入りをクリック
TOP     -down->    検索結果: 検索ボタンをクリック
検索結果: 結果一覧を表示


state 会員登録{
 [*]     ->  入力
 入力     ->  確認
 確認     ->  メール確認
 メール確認 -> 登録完了
}

state 会員情報変更{
 [*]     ->  入力
 入力     ->  確認
 確認     ->  メール確認
 メール確認 -> 登録完了
}

state カート内{
 [*]    -->    カート
 カート  ->    レジに進む
 レジに進む  ->  配送先確認
 配送先確認 ->   支払方法確認
 支払方法確認 -> 注文確認
 注文確認 ->  注文完了
}

@enduml
```
