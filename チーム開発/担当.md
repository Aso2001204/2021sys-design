# 管理者画面の機能 #
```uml
@startuml

管理者画面 --> NGワード
管理者画面:画面作成：終了
NGワード:ファイル名：ngword
NGワード:コントローラー：rootController@ngword
NGワード:ルーティング：/ngwordEdit

管理者画面 --> 公式のお題追加
公式のお題追加:ファイル名：addTitle
公式のお題追加:コントローラー：rootController@addTitle
公式のお題追加:ルーティング：/addTitle

管理者画面 --> ルームの全一覧
ルームの全一覧:ファイル名：roomAll
ルームの全一覧:コントローラー：rootController@roomAll
ルームの全一覧:ルーティング：/roomAll

管理者画面 --> チャット時間の変更
チャット時間の変更:ファイル名：ngword
チャット時間の変更:コントローラー：rootController@timeChange
チャット時間の変更:ルーティング：/timeChange
@enduml
```
